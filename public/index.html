Instructions

Calculate data right here in the address bar!

All take unlimited args except /math/pow/

  • sum: /math/sum/num/num/num
  • minus: /math/minus/minuend/subtrahend1/subtrahend2 (equals num1-num2-num3...)
  • multiply: /math/multiply/num1/num2/num3
  • divide: /math/divide/dividend/divisor1/divisor2/divisor3 (equals num1/num2/num3...)
  • pow: /math/pow/base/exponent

Analyze Sentences

  • wordCount: str/wordCount/type any string, including spaces
  • charCount: str/charCount/type any string to get character count
  • avgLen: str/avgLen/type any string and get the average word length

Age Calculator (in years)

  • age: age/mmddyyyy
